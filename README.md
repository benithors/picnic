# Picnic

A tiny macOS menu bar app for fast screenshots with simple markup (crop, arrows, text).

## Usage

1. Build and run:

   ```sh
   swift run
   ```

2. Grant Screen Recording permission when prompted.
3. Trigger capture:
   - `Ctrl` + `Help` (primary hotkey)
   - `Ctrl` + `Shift` + `5` (fallback hotkey)
4. Use the toolbar to crop, draw arrows, or add text.
5. Save or copy from the toolbar.

## Notes

- The app lives in the menu bar while running.
- Screen Recording permission is required for capture.
